The Array Toolkit extends functionality to Array Themes, providing portfolio items, testimonials, custom metaboxes and more.

==Description==

**Extend Your Array Theme**

The Array Toolkit provides support for portfolio items, testimonials, galleries, widgets and more to supported Array themes. Simply install the plugin and the theme will determine which features to activate based on theme support. See your theme's help file to determine whether or not you need the Array Toolkit.

==Frequently Asked Questions==

= What is this plugin and why do I need it? =

The Array Toolkit provides extra functionality to the collection of Array themes. The Toolkit adds custom post types (Portfolio and Testimonials), custom metaboxes and more. The plugin is not a requirement to use Array themes, but it will extend some themes to function as you see them in the demos.

= Where are the Toolkit settings? =

All options are activated automatically based on theme support so you don't need to change any settings. Check out your theme's help file to learn which features are available in your theme.

= Can I use this plugin with other themes? =

The Array Toolkit was developed to extend the functionality of Array themes specifically. It will not work with other themes out of the box.

==Installation==

This plugin can be installed directly from your site.

1. Log in and navigate to Plugins &rarr; Add New.
2. Type "Array Toolkit" into the Search input and click the "Search Widgets" button.
3. Locate the Array Toolkit in the list of search results and click "Install Now".
4. Click the "Activate Plugin" link at the bottom of the install screen.
5. The widgets will be available in Appearance &rarr; Widgets.

It can also be installed manually.

1. Download the "Array Toolkit" plugin from WordPress.org.
2. Unzip the package and move to your plugins directory.
3. Log into WordPress and navigate to the "Plugins" screen.
4. Locate "Array Toolkit" in the list and click the "Activate" link.
5. The widgets will be available in Appearance &rarr; Widgets.
